# XLI-CLI

Its a cli to create Angular2 project development environment.

# Features

 - Easy to install
 - Best for beginers 
 - Less setup step to create project
 - Only added required libraries to start with

# How to Use
 
 Run-
 > npm install -g xli-cli

 Then-
 > xli-cli
